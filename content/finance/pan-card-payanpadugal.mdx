## PAN Card பயன்பாடுகள்

PAN Card என்பது **Permanent Account Number card** ஆகும். இது Income Tax Department மூலம் வழங்கப்படுகிறது. PAN Card பல நிதி பரிவர்த்தனைகளுக்கு தேவை. இந்த கட்டுரையில், PAN Card பயன்பாடுகள் பற்றிய முழுமையான தகவல்களை பார்க்கலாம்.

## PAN Card என்றால் என்ன?

PAN Card என்பது 10 இலக்க alphanumeric number ஆகும். இது Income Tax Department மூலம் வழங்கப்படுகிறது. PAN Card ஒரு unique identification number ஆகும்.

### PAN Card Format

PAN Card format: `ABCDE1234F`

- **First 3 Characters**: Letters (A-Z)
- **4th Character**: Status (P for Person, C for Company, etc.)
- **5th Character**: First letter of surname
- **Next 4 Characters**: Numbers (0001-9999)
- **Last Character**: Letter (A-Z)

## PAN Card பயன்பாடுகள்

### Income Tax

PAN Card Income Tax filing இல் தேவை:

- **ITR Filing**: Income Tax return file செய்ய PAN Card mandatory
- **TDS**: TDS deduction and credit
- **Tax Payments**: Tax payments
- **Refunds**: Tax refunds

### Bank Accounts

PAN Card bank accounts திறக்க தேவை:

- **Savings Account**: ₹50,000 க்கு மேல் transactions
- **Current Account**: All current accounts
- **Fixed Deposit**: ₹50,000 க்கு மேல் FDs
- **Recurring Deposit**: ₹50,000 க்கு மேல் RDs

### Investments

PAN Card investments செய்ய தேவை:

- **Mutual Funds**: All mutual fund investments
- **Stocks**: Stock market investments
- **Bonds**: Government and corporate bonds
- **PPF**: Public Provident Fund
- **NSC**: National Savings Certificate

### Property Transactions

PAN Card property transactions இல் தேவை:

- **Property Buying**: Property purchase
- **Property Selling**: Property sale
- **Property Registration**: Property registration
- **Property Tax**: Property tax payments

### Loans

PAN Card loans எடுக்க தேவை:

- **Home Loan**: Home loan applications
- **Car Loan**: Car loan applications
- **Personal Loan**: Personal loan applications
- **Education Loan**: Education loan applications

### Credit Cards

PAN Card credit cards விண்ணப்பிக்க தேவை:

- **Credit Card Application**: All credit card applications
- **Credit Card Limit**: High limit credit cards

### Other Uses

- **Insurance**: High-value insurance policies
- **Dematerialized Account**: Demat account opening
- **Foreign Exchange**: Foreign exchange transactions
- **Cash Transactions**: High-value cash transactions

## PAN Card எவ்வாறு பெறுவது?

### முறை 1: Online Application (NSDL)

#### Step-by-Step Procedure:

1. **Portal Visit**
   - Browser இல் [https://www.tin-nsdl.com](https://www.tin-nsdl.com) visit செய்யவும்
   - "Services" → "PAN" → "Apply Online" select செய்யவும்

2. **Application Type Select**
   - "New PAN - Indian Citizen" select செய்யவும்
   - Form 49A select செய்யவும்

3. **Application Form Fill**
   - Personal details enter செய்யவும்:
     - Name (as per Aadhaar)
     - Date of Birth
     - Gender
     - Father's Name
     - Contact Details
   - Address details enter செய்யவும்
   - Aadhaar number enter செய்யவும் (optional, for e-KYC)

4. **Documents Upload**
   - Required documents upload செய்யவும்:
     - Photo (recent passport size)
     - Signature (scanned)
     - Identity Proof (Aadhaar Card)
     - Address Proof (Aadhaar Card, if address same)

5. **Payment**
   - Application fee: ₹93 (online)
   - Payment method select செய்யவும்:
     - Net Banking
     - Debit Card
     - Credit Card
     - UPI
   - Payment complete செய்யவும்

6. **Application Submit**
   - Form details verify செய்யவும்
   - "Submit" click செய்யவும்
   - Acknowledgement number note செய்யவும்

7. **PAN Card Receive**
   - Processing time: 15-20 working days
   - PAN Card by post receive செய்யலாம்
   - அல்லது e-PAN download செய்யலாம்

2. **Application Form Fill**
   - Similar process as NSDL
   - Form 49A fill செய்யவும்

3. **Documents Upload**
   - Required documents upload செய்யவும்

4. **Payment**
   - Application fee: ₹93 (online)
   - Payment complete செய்யவும்

5. **Application Submit**
   - Application submit செய்யவும்
   - Acknowledgement number note செய்யவும்

6. **PAN Card Receive**
   - Processing time: 15-20 working days
   - PAN Card receive செய்யலாம்

### முறை 3: Offline Application

#### Step-by-Step Procedure:

1. **Form Download**
   - NSDL ([https://www.tin-nsdl.com](https://www.tin-nsdl.com)) அல்லது UTIITSL ([https://www.utiitsl.com](https://www.utiitsl.com)) website visit செய்யவும்
   - Form 49A download செய்யவும்

2. **Form Fill**
   - Form carefully fill செய்யவும்
   - All required details enter செய்யவும்
   - Neat handwriting use செய்யவும்

3. **Documents Attach**
   - Required documents attach செய்யவும்:
     - Photo (recent passport size)
     - Signature
     - Identity Proof (self-attested copy)
     - Address Proof (self-attested copy)

4. **Application Submit**
   - NSDL/UTIITSL office இல் application submit செய்யவும்
   - அல்லது by post send செய்யவும்
   - Application fee: ₹101 (offline, by demand draft)

5. **PAN Card Receive**
   - Processing time: 15-20 working days
   - PAN Card by post receive செய்யலாம்

## PAN Card - Required Documents

### Mandatory Documents

1. **Photo**
   - Recent passport size photo
   - White background
   - Clear and visible

2. **Signature**
   - Signature on white paper
   - Scanned signature (for online)

3. **Identity Proof**
   - Aadhaar Card (preferred)
   - Voter ID Card
   - Driving License
   - Passport

4. **Address Proof**
   - Aadhaar Card (if address same as identity)
   - Voter ID Card
   - Electricity Bill
   - Bank Statement
   - Gas Bill

### Document Format Requirements

- **Photo**: JPG/JPEG, 20-50 KB, 3.5cm × 4.5cm
- **Signature**: JPG/JPEG, 10-20 KB
- **Other Documents**: PDF, 100-300 KB

## PAN Card மாற்றம்

### Name Change

Name change செய்தால், PAN Card update செய்ய வேண்டும்:

1. **Correction Application**
   - NSDL/UTIITSL portal இல் correction application submit செய்யவும்
   - Name change request செய்யவும்

2. **Documents**
   - Name change proof (Marriage Certificate, Gazette Notification)
   - Updated identity proof

3. **Fee**
   - Correction fee: ₹110 (online), ₹101 (offline)

### Address Change

Address change செய்தால், PAN Card update செய்ய வேண்டும்:

1. **Correction Application**
   - Address change request செய்யவும்

2. **Documents**
   - New address proof

3. **Fee**
   - Correction fee: ₹110 (online), ₹101 (offline)

## PAN Card - Common Issues மற்றும் Solutions

### Issue 1: Application Rejected

**Problem**: Application submit செய்த பிறகு rejected ஆகிறது.

**Solutions**:
- Rejection reason check செய்யவும்
- Documents verify செய்யவும்
- Missing documents upload/submit செய்யவும்
- Corrected application resubmit செய்யவும்

### Issue 2: PAN Card Not Received

**Problem**: Application approved ஆன பிறகு PAN Card receive ஆகவில்லை.

**Solutions**:
- Acknowledgement number மூலம் status check செய்யவும்
- e-PAN download செய்யலாம்
- NSDL/UTIITSL contact செய்யவும்
- Duplicate PAN Card apply செய்யலாம்

### Issue 3: PAN Card Lost/Damaged

**Problem**: PAN Card lost அல்லது damaged ஆகிறது.

**Solutions**:
- Duplicate PAN Card apply செய்யலாம்
- NSDL/UTIITSL portal இல் duplicate application submit செய்யவும்
- Fee: ₹50 (duplicate)
