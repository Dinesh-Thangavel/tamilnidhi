## ரேஷன் கார்டு திருத்தம் செய்வது எப்படி?

ரேஷன் கார்டு திருத்தம் செய்வது என்பது முக்கியமான செயல்முறையாகும். Name spelling mistakes, address change, family members add/remove, photo update போன்ற திருத்தங்கள் தேவைப்படலாம். இந்த கட்டுரையில், Tamil Nadu இல் ரேஷன் கார்டு திருத்தம் செய்வதற்கான முழுமையான வழிமுறைகளை பார்க்கலாம்.

## ரேஷன் கார்டு திருத்தம் - முக்கியத்துவம்

### ஏன் திருத்தம் தேவை?

- **Name Correction**: Spelling mistakes, name change
- **Address Update**: Current address update
- **Family Members**: New members add, removed members delete
- **Photo Update**: Old photo replace
- **Legal Compliance**: Accurate information for government services

## ரேஷன் கார்டு திருத்தம் வகைகள்

### 1. Name Correction

Name spelling mistakes, name change, அல்லது name update செய்யலாம்.

**Examples**:
- "Kumar" → "Kumaran"
- "Priya" → "Priyanka"
- Spelling mistakes correction

### 2. Address Change

Current address update செய்யலாம். New address proof தேவை.

**Examples**:
- Old address: 123, Main Street, Chennai
- New address: 456, Park Avenue, Chennai

### 3. Family Members Add/Remove

Family members add அல்லது remove செய்யலாம்.

**Add Members**:
- New born child
- Marriage (spouse add)
- Dependent relatives

**Remove Members**:
- Death
- Separation
- Independent living

### 4. Photo Update

Old photo replace செய்யலாம். Recent photo தேவை.

## ரேஷன் கார்டு திருத்தம் செயல்முறை

### முறை 1: Online (Tamil Nadu Food and Civil Supplies Portal)

#### Step-by-Step Procedure:

1. **Portal Visit**
   - Browser இல் [https://www.tnpds.gov.in](https://www.tnpds.gov.in) visit செய்யவும்
   - அல்லது [https://tnepds.gov.in](https://tnepds.gov.in) visit செய்யவும்

2. **Registration/Login**
   - New user என்றால், "Register" click செய்யவும்
   - Existing user என்றால், "Login" click செய்யவும்
   - Mobile number, password enter செய்யவும்
   - OTP verify செய்யவும்

3. **Correction/Update Section**
   - "Ration Card Services" → "Correction/Update" select செய்யவும்
   - "Apply for Correction" click செய்யவும்

4. **Ration Card Details Enter**
   - Ration Card Number enter செய்யவும்
   - Family Head Name enter செய்யவும்
   - Verify details

5. **Correction Type Select**
   - Correction type select செய்யவும்:
     - Name Correction
     - Address Change
     - Family Members Add/Remove
     - Photo Update

6. **New Details Enter**
   - Selected correction type க்கு ஏற்ப new details enter செய்யவும்
   - Example: Name correction → New name enter

7. **Documents Upload**
   - Required documents upload செய்யவும்:
     - Old ration card copy
     - Proof of correction (name proof, address proof, etc.)
     - Photo (photo update க்கு)
     - Supporting documents

8. **Application Submit**
   - Form details verify செய்யவும்
   - "Submit" click செய்யவும்
   - Application number note செய்யவும்

9. **Application Status Check**
   - Application number மூலம் status check செய்யலாம்
   - Processing time: 15-30 working days

10. **Updated Ration Card Download**
    - Application approved ஆன பிறகு, updated ration card download செய்யலாம்
    - PDF format இல் download செய்யலாம்

### முறை 2: Offline (Food and Civil Supplies Department Office)

#### Step-by-Step Procedure:

1. **Office Locate**
   - Nearest Food and Civil Supplies Department office locate செய்யவும்
   - District/Block level office visit செய்யவும்

2. **Application Form Collect**
   - Ration Card Correction Form collect செய்யவும்
   - அல்லது [https://www.tnpds.gov.in](https://www.tnpds.gov.in) இல் download செய்யவும்

3. **Form Fill**
   - Form fill செய்யவும்:
     - Ration Card Number
     - Family Head Name
     - Correction Type
     - New Details
     - Contact Information

4. **Documents Prepare**
   - Required documents prepare செய்யவும்:
     - Original old ration card
     - Self-attested copy of old ration card
     - Proof of correction
     - Supporting documents
     - Recent photo (photo update க்கு)

5. **Office Visit**
   - Office visit செய்து, form மற்றும் documents submit செய்யவும்

6. **Verification**
   - Officer verification செய்யலாம்
   - Additional documents request செய்யலாம் (if needed)

7. **Acknowledgement Receipt**
   - Acknowledgement receipt collect செய்யவும்
   - Application number note செய்யவும்

8. **Status Check**
   - Application number மூலம் status check செய்யலாம்
   - Processing time: 15-30 working days

9. **Updated Ration Card Collect**
   - Application approved ஆன பிறகு, office இல் updated ration card collect செய்யலாம்
   - அல்லது online download செய்யலாம்

## ரேஷன் கார்டு திருத்தம் - Common Issues மற்றும் Solutions

### Issue 1: Application Rejected

**Problem**: Application submit செய்த பிறகு rejected ஆகிறது.

**Solutions**:
- Rejection reason check செய்யவும்
- Documents verify செய்யவும்
- Missing documents upload/submit செய்யவும்
- Corrected application resubmit செய்யவும்
- Helpdesk உடன் தொடர்பு கொள்ளவும்

### Issue 2: Documents Not Uploading

**Problem**: Documents upload செய்யும்போது error வருகிறது.

**Solutions**:
- File size 2MB க்கு கீழே இருக்கிறதா check செய்யவும்
- File format (PDF/JPG/PNG) correct ஆக இருக்கிறதா verify செய்யவும்
- Internet connection stable ஆக இருக்கிறதா check செய்யவும்
- Different browser try செய்யவும்
- File compress செய்து upload செய்யவும்

### Issue 3: Application Status Not Updating

**Problem**: Application submit செய்த பிறகு status update ஆகவில்லை.

**Solutions**:
- Application number note செய்யவும்
- Few days wait செய்யவும் (processing time)
- Portal இல் status check செய்யவும்
- Helpdesk உடன் application number share செய்து status check செய்யவும்
- Office visit செய்து status verify செய்யவும்

### Issue 4: Updated Ration Card Not Received

**Problem**: Application approved ஆன பிறகு updated ration card receive ஆகவில்லை.

**Solutions**:
- Portal இல் download செய்யவும்
- Application number மூலம் download செய்யவும்
- Offline application செய்திருந்தால், office இல் collect செய்யலாம்
- Helpdesk உடன் தொடர்பு கொள்ளவும்

### Issue 5: Multiple Corrections

**Problem**: Multiple corrections (name + address) செய்ய வேண்டும்.

**Solutions**:
- Separate applications submit செய்யலாம்
- அல்லது single application இல் multiple corrections mention செய்யலாம்
- All required documents upload/submit செய்யவும்
- Each correction க்கு separate fee charge செய்யப்படலாம்

### என்ன திருத்தலாம்?

Name, address, family members (add/remove), photo திருத்தலாம். Multiple corrections single application இல் செய்யலாம்.

### தேவையான ஆவணங்கள் என்ன?

Correction type பொறுத்து documents vary ஆகும்:
- **Name Correction**: Old ration card, name proof (Aadhaar/Voter ID), supporting documents
- **Address Change**: Old ration card, address proof (Electricity Bill/Gas Bill/Bank Statement), identity proof
- **Family Members Add**: Old ration card, new member documents (Birth Certificate/Marriage Certificate), identity proof, photo
- **Family Members Remove**: Old ration card, removal proof (Death Certificate/Separation Certificate)
- **Photo Update**: Old ration card, recent photo, identity proof

### எவ்வளவு நேரம் எடுக்கும்?

Processing time: 15-30 working days. Application number மூலம் status check செய்யலாம்.

### Fee எவ்வளவு?

Application fee: ₹10 - ₹50 (varies by correction type). Payment methods: Online payment, Cash (offline).

### Application Rejected ஆனால் என்ன செய்வது?

Application rejected ஆனால்:
1. Rejection reason check செய்யவும்
2. Missing documents upload/submit செய்யவும்
3. Corrected application resubmit செய்யவும்

### Updated Ration Card எவ்வாறு பெறுவது?

### Multiple Corrections செய்யலாமா?

ஆம், multiple corrections செய்யலாம். Single application இல் multiple corrections mention செய்யலாம் அல்லது separate applications submit செய்யலாம். Each correction க்கு separate fee charge செய்யப்படலாம்.

### Family Members எவ்வளவு add செய்யலாம்?

Family members add செய்வது family size limit பொறுத்து vary ஆகும். Generally, immediate family members (spouse, children, dependent parents) add செய்யலாம். Maximum family size limit check செய்யவும்.
