## UMANG App Registration - Step-by-Step Procedure

### Step-by-Step Registration:

1. **App Download**
   - Google Play Store அல்லது Apple App Store இல் "UMANG" search செய்யவும்
   - Official UMANG app download செய்யவும்

2. **App Open**
   - App open செய்யவும்
   - "Register" அல்லது "Sign Up" click செய்யவும்

3. **Registration Method Select**
   - Registration method select செய்யவும்:
     - Mobile Number
     - Aadhaar Number
     - Email ID

4. **Mobile Number Registration** (Recommended)
   - Mobile number enter செய்யவும்
   - OTP request செய்யவும்
   - OTP enter செய்து verify செய்யவும்

5. **Aadhaar Registration** (Optional)
   - Aadhaar number enter செய்யவும்
   - OTP verify செய்யவும்
   - Aadhaar link செய்யலாம்

6. **Profile Setup**
   - Name enter செய்யவும்
   - Email ID enter செய்யவும் (optional)
   - Profile complete செய்யவும்

7. **Registration Complete**
   - Registration successful
   - Services access செய்யலாம்

## UMANG App - Common Issues மற்றும் Solutions

### Issue 1: OTP Not Received

**Problem**: OTP request செய்த பிறகு OTP receive ஆகவில்லை.

**Solutions**:
- Mobile number சரியாக enter செய்துள்ளீர்களா verify செய்யவும்
- Mobile network signal check செய்யவும்
- SMS inbox full ஆக இருக்கிறதா check செய்யவும்
- Few minutes wait செய்து retry செய்யவும்
- "Resend OTP" option use செய்யவும்

### Issue 2: Services Not Loading

**Problem**: Services load ஆகவில்லை.

**Solutions**:
- Internet connection check செய்யவும்
- App update செய்யவும்
- App restart செய்யவும்
- Cache clear செய்யவும்
- Different network try செய்யவும்

### Issue 3: Payment Failed

**Problem**: Payment attempt செய்த பிறகு failed ஆகிறது.

**Solutions**:
- Bank account/credit card இல் sufficient balance இருக்கிறதா check செய்யவும்
- Internet connection stable ஆக இருக்கிறதா verify செய்யவும்
- Payment gateway issue இருந்தால், few minutes கழித்து retry செய்யவும்
- Different payment method try செய்யவும்
- Bank உடன் தொடர்பு கொண்டு transaction status check செய்யவும்

### Issue 4: Certificate Not Downloading

**Problem**: Certificate download ஆகவில்லை.

**Solutions**:
- Internet connection check செய்யவும்
- Storage space available இருக்கிறதா check செய்யவும்
- PDF reader installed ஆக இருக்கிறதா verify செய்யவும்
- Different browser/device try செய்யவும்
- Helpdesk உடன் தொடர்பு கொள்ளவும்
