## ஆதார் முகவரி மாற்றம் செய்வது எப்படி?

ஆதார் முகவரி மாற்றம் செய்வது என்பது முக்கியமான செயல்முறையாகும். முகவரி மாற்றம், புதிய முகவரி update, அல்லது முகவரி correction செய்ய தேவைப்படலாம். இந்த கட்டுரையில், ஆதார் முகவரி மாற்றம் செய்வதற்கான முழுமையான வழிமுறைகளை பார்க்கலாம்.

## ஆதார் முகவரி மாற்றம் - முக்கியத்துவம்

### முகவரி மாற்றம் ஏன் தேவை?

- **Current Address Update**: தற்போதைய முகவரி update செய்ய
- **Address Proof**: பல services க்கு address proof ஆக பயன்படும்
- **Government Schemes**: Government schemes apply செய்ய தேவை
- **Bank Accounts**: Bank accounts update செய்ய தேவை
- **Legal Purposes**: Legal matters இல் address proof ஆக பயன்படும்

## ஆதார் முகவரி மாற்றம் வழிகள்

### முறை 1: Online (UIDAI Portal)

UIDAI (Unique Identification Authority of India) official portal மூலம் online முகவரி மாற்றம் செய்யலாம்.

#### Step-by-Step Procedure:

1. **Portal Visit**
   - Browser இல் [https://uidai.gov.in](https://uidai.gov.in) அல்லது [https://ssup.uidai.gov.in](https://ssup.uidai.gov.in) visit செய்யவும்

2. **Update Aadhaar Section**
   - "My Aadhaar" → "Update Your Aadhaar" select செய்யவும்
   - "Update Address Online" click செய்யவும்

3. **Aadhaar Number Enter**
   - உங்கள் 12-digit Aadhaar number enter செய்யவும்
   - Captcha enter செய்யவும்
   - "Send OTP" click செய்யவும்

4. **OTP Verification**
   - Registered mobile number இல் OTP receive செய்யவும்
   - OTP enter செய்து verify செய்யவும்

5. **Address Update Form**
   - New address enter செய்யவும்:
     - House/Flat No
     - Street/Area
     - Landmark
     - City/Town
     - State
     - PIN Code
   - Address exactly as per address proof enter செய்யவும்

6. **Address Proof Upload**
   - Valid address proof document select செய்யவும்
   - Document upload செய்யவும்
   - Document format: PDF, JPG, PNG (Max 2MB)

7. **Payment**
   - Update fee: ₹50
   - Payment method select செய்யவும் (Debit Card/Credit Card/Net Banking/UPI)
   - Payment complete செய்யவும்

8. **Update Request Number**
   - Update Request Number (URN) receive செய்யவும்
   - URN note செய்யவும் (status check க்கு)

9. **Status Check**
   - URN மூலம் status check செய்யலாம்
   - Processing time: 15-30 days

10. **Updated Aadhaar Download**
    - Update approved ஆன பிறகு, updated Aadhaar download செய்யலாம்
    - e-Aadhaar PDF download செய்யலாம்

### முறை 2: Offline (Aadhaar Enrolment Centre)

Online apply செய்ய முடியாத நிலையில், offline மூலம் முகவரி மாற்றம் செய்யலாம்.

#### Step-by-Step Procedure:

1. **Enrolment Centre Locate**
   - Nearest Aadhaar Enrolment Centre locate செய்யவும்
   - [https://appointments.uidai.gov.in](https://appointments.uidai.gov.in) மூலம் centre find செய்யலாம்

2. **Appointment Book** (Optional)
   - Online appointment book செய்யலாம்
   - Walk-in கூட allowed (depends on centre)

3. **Form Download/Fill**
   - Aadhaar Update/Correction Form download செய்யவும்
   - அல்லது centre இல் form collect செய்யவும்
   - Form fill செய்யவும்:
     - Aadhaar Number
     - Name
     - Address (new address)
     - Mobile Number
     - Email (optional)

4. **Documents Prepare**
   - Original address proof document
   - Self-attested copy of address proof
   - Old Aadhaar card copy

5. **Centre Visit**
   - Enrolment Centre visit செய்யவும்
   - Form மற்றும் documents submit செய்யவும்

6. **Biometric Verification**
   - Fingerprints scan செய்யவும்
   - Iris scan செய்யவும்
   - Photo capture செய்யவும்

7. **Payment**
   - Update fee: ₹50
   - Cash payment செய்யவும்

8. **Acknowledgement Receipt**
   - Acknowledgement receipt collect செய்யவும்
   - Update Request Number (URN) note செய்யவும்

9. **Status Check**
   - URN மூலம் status check செய்யலாம்
   - Processing time: 15-30 days

10. **Updated Aadhaar Collect**
    - Update approved ஆன பிறகு, updated Aadhaar collect செய்யலாம்
    - Centre இல் collect செய்யலாம் அல்லது e-Aadhaar download செய்யலாம்

## தேவையான ஆவணங்கள் (Address Proof)

### Valid Address Proof Documents

1. **Electricity Bill**
   - Latest electricity bill (not older than 3 months)
   - Name மற்றும் address clearly visible

2. **Gas Bill**
   - Latest gas connection bill
   - Name மற்றும் address clearly visible

3. **Bank Statement/Passbook**
   - Bank statement with address
   - Passbook with address
   - Not older than 3 months

4. **Rent Agreement**
   - Registered rent agreement
   - Landlord's address proof attached

5. **Voter ID Card**
   - Voter ID card with current address

6. **Driving License**
   - Driving License with current address

7. **Passport**
   - Passport with current address

8. **Property Documents**
   - Property registration documents
   - Property tax receipt

9. **Government Documents**
   - Government issued documents with address
   - PSU documents with address

### Document Requirements

- **Original Document**: Offline application க்கு original document தேவை
- **Self-Attested Copy**: Self-attested copy submit செய்யவும்
- **Clear Copy**: Document clear ஆக இருக்க வேண்டும்
- **Valid Document**: Document valid ஆக இருக்க வேண்டும் (not expired)
- **Name Match**: Document இல் name, Aadhaar இல் name match ஆக இருக்க வேண்டும்

## ஆதார் முகவரி மாற்றம் - Common Issues மற்றும் Solutions

### Issue 1: OTP Not Received

**Problem**: OTP request செய்த பிறகு OTP receive ஆகவில்லை.

**Solutions**:
- Registered mobile number சரியாக enter செய்துள்ளீர்களா verify செய்யவும்
- Mobile network signal check செய்யவும்
- SMS inbox full ஆக இருக்கிறதா check செய்யவும்
- Few minutes wait செய்து retry செய்யவும்
- "Resend OTP" option use செய்யவும்
- Different mobile number registered ஆக இருந்தால், mobile number update செய்யவும்

### Issue 2: Address Proof Rejected

**Problem**: Address proof upload செய்த பிறகு rejected ஆகிறது.

**Solutions**:
- Address proof valid ஆக இருக்கிறதா verify செய்யவும் (not older than 3 months)
- Document clear ஆக இருக்கிறதா check செய்யவும்
- Name, Aadhaar name match ஆக இருக்கிறதா verify செய்யவும்
- Address, new address match ஆக இருக்கிறதா check செய்யவும்
- Different valid address proof try செய்யவும்
- Document format (PDF/JPG/PNG) correct ஆக இருக்கிறதா verify செய்யவும்

### Issue 3: Payment Failed

**Problem**: Payment attempt செய்த பிறகு failed ஆகிறது.

**Solutions**:
- Bank account/credit card இல் sufficient balance இருக்கிறதா check செய்யவும்
- Internet connection stable ஆக இருக்கிறதா verify செய்யவும்
- Payment gateway issue இருந்தால், few minutes கழித்து retry செய்யவும்
- Different payment method try செய்யவும்
- Bank உடன் தொடர்பு கொண்டு transaction status check செய்யவும்

### Issue 4: Update Status Not Changing

**Problem**: Application submit செய்த பிறகு status update ஆகவில்லை.

**Solutions**:
- URN note செய்யவும்
- Processing time (15-30 days) wait செய்யவும்
- URN மூலம் status check செய்யவும்
- Helpdesk (1947) உடன் URN share செய்து status check செய்யவும்
- Email/SMS updates check செய்யவும்

### Issue 5: Updated Aadhaar Not Received

**Problem**: Update approved ஆன பிறகு updated Aadhaar receive ஆகவில்லை.

**Solutions**:
- e-Aadhaar download செய்யவும் ([https://eaadhaar.uidai.gov.in](https://eaadhaar.uidai.gov.in))
- Aadhaar number மற்றும் registered mobile number enter செய்யவும்
- OTP verify செய்து e-Aadhaar download செய்யவும்
- Offline application செய்திருந்தால், Enrolment Centre இல் collect செய்யலாம்
- Helpdesk (1947) உடன் தொடர்பு கொள்ளவும்

4. Offline application செய்திருந்தால், Enrolment Centre இல் collect செய்யலாம்

### Mobile Number Update செய்யலாமா?

### Multiple Address Updates செய்யலாமா?

ஆம், multiple address updates செய்யலாம். ஒவ்வொரு update க்கும் separate application submit செய்ய வேண்டும். Each update க்கு ₹50 fee charge செய்யப்படும்.
