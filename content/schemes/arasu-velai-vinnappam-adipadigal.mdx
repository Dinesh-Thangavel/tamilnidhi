## அரசு வேலை விண்ணப்ப அடிப்படைகள்

அரசு வேலை விண்ணப்ப அடிப்படைகள் என்பது அரசு வேலைக்கு விண்ணப்பிக்க புரிந்து கொள்ள வேண்டிய fundamental concepts ஆகும். இந்த கட்டுரையில், அரசு வேலை விண்ணப்பம் பற்றிய முழுமையான தகவல்களை பார்க்கலாம்.

## அரசு வேலை விண்ணப்பம் - Overview

அரசு வேலை விண்ணப்பம் என்பது central government, state government, அல்லது public sector undertakings (PSUs) இல் வேலைக்கு apply செய்வது ஆகும்.

### அரசு வேலை வகைகள்

- **Central Government Jobs**: Central government departments, ministries
- **State Government Jobs**: State government departments
- **PSU Jobs**: Public Sector Undertakings (Banks, Railways, etc.)
- **Defense Jobs**: Army, Navy, Air Force
- **Police Jobs**: Police department, security forces

## அரசு வேலை விண்ணப்பம் - Step-by-Step Procedure

### முறை 1: Online Application

#### Step-by-Step Procedure:

1. **Job Notification Check**
   - Government job portals visit செய்யவும்:
     - [https://www.sarkariresult.com](https://www.sarkariresult.com)
     - [https://www.freejobalert.com](https://www.freejobalert.com)
     - [https://www.india.gov.in/jobs](https://www.india.gov.in/jobs)
     - State government portals
   - Job notifications regularly check செய்யவும்

2. **Job Details Read**
   - Job notification carefully read செய்யவும்
   - Eligibility criteria check செய்யவும்
   - Application deadline note செய்யவும்
   - Required documents list check செய்யவும்

3. **Registration**
   - Official portal visit செய்யவும்
   - "Register" click செய்யவும்
   - Personal details enter செய்யவும்:
     - Name, Date of Birth, Email, Mobile
     - Aadhaar Number, PAN Number
   - Username and Password create செய்யவும்
   - OTP verify செய்யவும்

4. **Application Form Fill**
   - "Apply Online" click செய்யவும்
   - Application form fill செய்யவும்:
     - Personal Information
     - Educational Qualifications
     - Work Experience (if applicable)
     - Category (General/SC/ST/OBC/EWS)
     - Preferences

5. **Documents Upload**
   - Required documents upload செய்யவும்:
     - Photo (recent passport size)
     - Signature (scanned)
     - Educational Certificates
     - Age Proof
     - Category Certificate (if applicable)
     - Experience Certificates (if applicable)

6. **Application Fee Payment**
   - Application fee calculate செய்யவும்
   - Payment method select செய்யவும்:
     - Online Payment (Net Banking, Debit Card, Credit Card, UPI)
     - Offline Payment (Challan)
   - Payment complete செய்யவும்
   - Payment receipt save செய்யவும்

7. **Application Submit**
   - Form details verify செய்யவும்
   - "Submit" click செய்யவும்
   - Application number note செய்யவும்
   - Application confirmation print செய்யவும்

8. **Application Status Check**
   - Application number மூலம் status check செய்யலாம்
   - Admit card download (when available)
   - Exam/Interview updates check செய்யலாம்

### முறை 2: Offline Application

#### Step-by-Step Procedure:

1. **Application Form Collect**
   - Official notification download செய்யவும்
   - Application form download செய்யவும்
   - அல்லது designated office இல் form collect செய்யவும்

2. **Form Fill**
   - Form carefully fill செய்யவும்
   - All required details enter செய்யவும்
   - Neat handwriting use செய்யவும்

3. **Documents Attach**
   - Required documents attach செய்யவும்:
     - Self-attested copies
     - Photos
     - Certificates

4. **Application Submit**
   - Designated office இல் application submit செய்யவும்
   - Application number receive செய்யவும்
   - Acknowledgement receipt collect செய்யவும்

## அரசு வேலை விண்ணப்பம் - Eligibility Criteria

### Educational Qualification

- **Minimum Qualification**: Job notification இல் mentioned minimum qualification
- **Degree/Diploma**: Relevant degree or diploma
- **Percentage**: Minimum percentage requirement (if specified)
- **Specialization**: Specific specialization (if required)

### Age Limit

- **Minimum Age**: Usually 18 years
- **Maximum Age**: Varies by category:
  - General: 30-35 years
  - SC/ST: 5 years relaxation
  - OBC: 3 years relaxation
  - PWD: 10 years relaxation
- **Age Calculation**: As on specific date mentioned in notification

### Experience

- **Work Experience**: Some jobs require work experience
- **Experience Certificate**: Experience certificates required
- **Relevant Experience**: Relevant field experience preferred

### Category

- **General**: General category
- **SC/ST**: Scheduled Caste/Scheduled Tribe
- **OBC**: Other Backward Classes
- **EWS**: Economically Weaker Section
- **PWD**: Persons with Disabilities

## அரசு வேலை விண்ணப்பம் - Common Issues மற்றும் Solutions

### Issue 1: Registration Failed

**Problem**: Registration complete ஆகவில்லை.

**Solutions**:
- Email/Mobile already registered ஆக இருந்தால், login செய்யவும்
- OTP correctly enter செய்யவும்
- Internet connection stable ஆக இருக்கிறதா check செய்யவும்
- Different browser try செய்யவும்

### Issue 2: Documents Not Uploading

**Problem**: Documents upload fail ஆகிறது.

**Solutions**:
- File size limit check செய்யவும் (usually 200KB-500KB)
- File format (JPG/PDF) correct ஆக இருக்கிறதா verify செய்யவும்
- File compress செய்யவும்
- Different browser try செய்யவும்

### Issue 3: Payment Failed

**Problem**: Payment attempt செய்த பிறகு failed ஆகிறது.

**Solutions**:
- Bank account/credit card இல் sufficient balance இருக்கிறதா check செய்யவும்
- Payment gateway issue இருந்தால், few minutes கழித்து retry செய்யவும்
- Different payment method try செய்யவும்
- Bank உடன் தொடர்பு கொண்டு transaction status check செய்யவும்
