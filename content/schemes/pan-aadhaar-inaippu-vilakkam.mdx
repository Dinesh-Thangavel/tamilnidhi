## PAN–Aadhaar இணைப்பு செயல்முறை

### முறை 1: Online (Income Tax Portal)

Income Tax portal மூலம் online PAN–Aadhaar இணைப்பு செய்யலாம்.

#### Step-by-Step Procedure:

1. **Portal Visit**
   - Browser இல் [https://www.incometax.gov.in](https://www.incometax.gov.in) visit செய்யவும்

2. **Login**
   - "Login" click செய்யவும்
   - PAN card number enter செய்யவும்
   - Password enter செய்யவும்
   - Captcha enter செய்யவும்
   - "Login" click செய்யவும்

3. **Link Aadhaar Section**
   - "Profile" → "Link Aadhaar" select செய்யவும்
   - அல்லது "Quick Links" → "Link Aadhaar" select செய்யவும்

4. **Aadhaar Details Enter**
   - 12-digit Aadhaar number enter செய்யவும்
   - Name as per Aadhaar enter செய்யவும் (exactly as in Aadhaar)
   - Date of Birth enter செய்யவும் (as per Aadhaar)
   - Gender select செய்யவும்

5. **Verification**
   - Details verify செய்யவும்
   - "Link Aadhaar" click செய்யவும்

6. **OTP Verification**
   - Registered mobile number இல் OTP receive செய்யவும்
   - OTP enter செய்து verify செய்யவும்

7. **Link Complete**
   - Link successful message பெறவும்
   - Link status verify செய்யலாம்

### முறை 2: SMS

SMS மூலம் PAN–Aadhaar இணைப்பு செய்யலாம்.

#### Step-by-Step Procedure:

1. **SMS Format**
   - SMS format: `UIDAI-Aadhaar <12-digit Aadhaar> <10-character PAN>`
   - Example: `UIDAI-Aadhaar 123456789012 ABCDE1234F`

2. **SMS Send**
   - Registered mobile number இல் SMS send செய்யவும்
   - To: `567678` அல்லது `56161`

3. **Confirmation**
   - Confirmation SMS receive செய்யலாம்
   - Link status verify செய்யலாம்

### முறை 3: Offline (NSDL/UTIITSL)

Offline மூலம் PAN–Aadhaar இணைப்பு செய்யலாம்.

#### Step-by-Step Procedure:

1. **Form Download**
   - NSDL ([https://www.tin-nsdl.com](https://www.tin-nsdl.com)) அல்லது UTIITSL ([https://www.utiitsl.com](https://www.utiitsl.com)) website visit செய்யவும்
   - PAN–Aadhaar linking form download செய்யவும்

2. **Form Fill**
   - Form fill செய்யவும்:
     - PAN number
     - Aadhaar number
     - Name
     - Date of Birth
     - Contact details

3. **Documents Attach**
   - PAN card copy
   - Aadhaar card copy
   - Self-attested copies

4. **Form Submit**
   - NSDL/UTIITSL office இல் form submit செய்யவும்
   - அல்லது by post send செய்யவும்

5. **Processing**
   - Processing time: 7-15 working days
   - Link status verify செய்யலாம்

## PAN–Aadhaar இணைப்பு - Common Issues மற்றும் Solutions

### Issue 1: Name Mismatch

**Problem**: PAN இல் name, Aadhaar இல் name match ஆகவில்லை.

**Solutions**:
- Name exactly as per Aadhaar enter செய்யவும்
- Middle name/surname correctly enter செய்யவும்
- Spelling exactly match ஆக இருக்க வேண்டும்
- Aadhaar name correction செய்யலாம் (if needed)
- PAN name correction செய்யலாம் (if needed)

### Issue 2: Date of Birth Mismatch

**Problem**: PAN இல் DOB, Aadhaar இல் DOB match ஆகவில்லை.

**Solutions**:
- Date of Birth exactly as per Aadhaar enter செய்யவும்
- Format correctly enter செய்யவும் (DD/MM/YYYY)
- Aadhaar DOB correction செய்யலாம் (if needed)
- PAN DOB correction செய்யலாம் (if needed)

### Issue 3: OTP Not Received

**Problem**: OTP request செய்த பிறகு OTP receive ஆகவில்லை.

**Solutions**:
- Registered mobile number சரியாக enter செய்துள்ளீர்களா verify செய்யவும்
- Mobile number Aadhaar இல் registered ஆக இருக்கிறதா verify செய்யவும்
- Mobile network signal check செய்யவும்
- SMS inbox full ஆக இருக்கிறதா check செய்யவும்
- Few minutes wait செய்து retry செய்யவும்
- "Resend OTP" option use செய்யவும்

### Issue 4: Link Failed

**Problem**: Link attempt செய்த பிறகு failed ஆகிறது.

**Solutions**:
- Name, DOB, Gender exactly match ஆக enter செய்துள்ளீர்களா verify செய்யவும்
- Aadhaar number correctly enter செய்துள்ளீர்களா check செய்யவும்
- PAN number correctly enter செய்துள்ளீர்களா verify செய்யவும்
- Already linked ஆக இருந்தால், status check செய்யவும்
- Helpdesk உடன் தொடர்பு கொள்ளவும்

### Issue 5: Already Linked

**Problem**: Already linked message வருகிறது.

**Solutions**:
- Link status verify செய்யவும்
- Income Tax portal இல் status check செய்யவும்
- Already linked ஆக இருந்தால், no action needed
- Different PAN/Aadhaar use செய்யலாம் (if applicable)
