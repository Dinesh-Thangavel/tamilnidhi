## DigiLocker என்றால் என்ன?

DigiLocker என்பது இந்திய அரசின் digital documents storage service ஆகும். இது ஒரு secure digital locker ஆகும், இதில் நீங்கள் உங்கள் important documents store செய்யலாம் மற்றும் access செய்யலாம். இந்த கட்டுரையில், DigiLocker பற்றிய முழுமையான தகவல்களை பார்க்கலாம்.

## DigiLocker Registration - Step-by-Step Procedure

### முறை 1: Website Registration

#### Step-by-Step Procedure:

1. **Website Visit**
   - Browser இல் [https://digilocker.gov.in](https://digilocker.gov.in) visit செய்யவும்

2. **Sign Up**
   - "Sign Up" button click செய்யவும்
   - "Sign Up with Aadhaar" select செய்யவும்

3. **Aadhaar Number Enter**
   - 12-digit Aadhaar number enter செய்யவும்
   - Captcha enter செய்யவும்
   - "Continue" click செய்யவும்

4. **OTP Verification**
   - Registered mobile number இல் OTP receive செய்யவும்
   - OTP enter செய்து verify செய்யவும்

5. **Username and Password**
   - Username choose செய்யவும்
   - Password set செய்யவும் (strong password)
   - Security questions answer செய்யவும்

6. **Registration Complete**
   - Registration successful message பெறவும்
   - Login செய்யலாம்

## DigiLocker - Documents Access Methods

## DigiLocker - Common Issues மற்றும் Solutions

### Issue 1: OTP Not Received

**Problem**: OTP request செய்த பிறகு OTP receive ஆகவில்லை.

**Solutions**:
- Registered mobile number சரியாக enter செய்துள்ளீர்களா verify செய்யவும்
- Mobile network signal check செய்யவும்
- SMS inbox full ஆக இருக்கிறதா check செய்யவும்
- Few minutes wait செய்து retry செய்யவும்
- "Resend OTP" option use செய்யவும்
- Different mobile number registered ஆக இருந்தால், mobile number update செய்யவும்

### Issue 2: Documents Not Fetching

**Problem**: Issued documents fetch ஆகவில்லை.

**Solutions**:
- Department select correctly செய்துள்ளீர்களா verify செய்யவும்
- Document type select correctly செய்துள்ளீர்களா check செய்யவும்
- Aadhaar number linked ஆக இருக்கிறதா verify செய்யவும்
- Few minutes wait செய்து retry செய்யவும்
- Helpdesk உடன் தொடர்பு கொள்ளவும்

### Issue 3: Upload Failed

**Problem**: Documents upload fail ஆகிறது.

**Solutions**:
- File size 10MB க்கு கீழே இருக்கிறதா check செய்யவும்
- File format (PDF/JPG/PNG) correct ஆக இருக்கிறதா verify செய்யவும்
- Internet connection stable ஆக இருக்கிறதா check செய்யவும்
- Different browser try செய்யவும்
- File compress செய்து upload செய்யவும்

### Issue 4: Login Issues

**Problem**: Login செய்ய முடியவில்லை.

**Solutions**:
- Username/Password சரியாக enter செய்துள்ளீர்களா verify செய்யவும்
- "Forgot Password" option use செய்யவும்
- Aadhaar number மூலம் login try செய்யவும்
- Helpdesk உடன் தொடர்பு கொள்ளவும்
