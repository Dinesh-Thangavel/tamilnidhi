- **Transparency**: Application status track செய்யலாம்

## Online Services Registration - Step-by-Step Procedure

### Tamil Nadu e-District Portal Registration

#### Step-by-Step Procedure:

1. **Portal Visit**
   - Browser இல் [https://edistrict.tn.gov.in](https://edistrict.tn.gov.in) visit செய்யவும்

2. **Registration**
   - "Register" அல்லது "Sign Up" click செய்யவும்

3. **Registration Method Select**
   - Mobile number registration select செய்யவும்
   - அல்லது Aadhaar-based registration select செய்யவும்

4. **Mobile Number Registration**
   - Mobile number enter செய்யவும்
   - Email ID enter செய்யவும் (optional)
   - Password set செய்யவும் (strong password)
   - OTP request செய்யவும்

5. **OTP Verification**
   - Registered mobile number இல் OTP receive செய்யவும்
   - OTP enter செய்து verify செய்யவும்

6. **Profile Setup**
   - Name enter செய்யவும்
   - Address enter செய்யவும் (optional)
   - Profile complete செய்யவும்

7. **Registration Complete**
   - Registration successful message பெறவும்
   - Login செய்யலாம்

### Aadhaar-based Registration

1. **Aadhaar Registration**
   - "Register with Aadhaar" select செய்யவும்
   - 12-digit Aadhaar number enter செய்யவும்
   - OTP verify செய்யவும்

2. **Profile Setup**
   - Aadhaar details automatically fetch ஆகும்
   - Additional details enter செய்யவும் (if required)

3. **Registration Complete**
   - Registration successful
   - Login செய்யலாம்

## Online Services Login - Step-by-Step Procedure

### Login Procedure:

1. **Portal Visit**
   - Portal website visit செய்யவும்

2. **Login Section**
   - "Login" அல்லது "Sign In" click செய்யவும்

3. **Credentials Enter**
   - Mobile number/Aadhaar number enter செய்யவும்
   - Password enter செய்யவும்
   - Captcha enter செய்யவும் (if required)

4. **Login**
   - "Login" click செய்யவும்
   - OTP verify செய்யலாம் (if required)

5. **Dashboard Access**
   - Dashboard access செய்யலாம்
   - Services access செய்யலாம்

## Online Services Access - Step-by-Step Procedure

### Service Application:

1. **Services Section**
   - "Services" அல்லது "Apply Online" section select செய்யவும்

2. **Service Select**
   - Required service select செய்யவும் (e.g., Income Certificate, Birth Certificate)

3. **Application Form**
   - Application form fill செய்யவும்
   - Required details enter செய்யவும்

4. **Documents Upload**
   - Required documents upload செய்யவும்
   - Document format and size verify செய்யவும்

5. **Application Submit**
   - Form details verify செய்யவும்
   - "Submit" click செய்யவும்
   - Application number note செய்யவும்

6. **Payment** (if required)
   - Payment method select செய்யவும்
   - Payment complete செய்யவும்

7. **Status Check**
   - Application number மூலம் status check செய்யலாம்
   - Regular updates check செய்யலாம்

8. **Certificate/Service Receive**
   - Application approved ஆன பிறகு, certificate/service receive செய்யலாம்
   - Download/Collect செய்யலாம்

## Online Services - Common Issues மற்றும் Solutions

### Issue 1: Registration Failed

**Problem**: Registration complete ஆகவில்லை.

**Solutions**:
- Mobile number/Aadhaar number சரியாக enter செய்துள்ளீர்களா verify செய்யவும்
- OTP correctly enter செய்துள்ளீர்களா check செய்யவும்
- Internet connection stable ஆக இருக்கிறதா verify செய்யவும்
- Different browser try செய்யவும்
- Helpdesk உடன் தொடர்பு கொள்ளவும்

### Issue 2: Login Issues

**Problem**: Login செய்ய முடியவில்லை.

**Solutions**:
- Username/Password சரியாக enter செய்துள்ளீர்களா verify செய்யவும்
- "Forgot Password" option use செய்யவும்
- OTP correctly enter செய்துள்ளீர்களா check செய்யவும்
- Account locked ஆக இருந்தால், unlock request செய்யவும்
- Helpdesk உடன் தொடர்பு கொள்ளவும்

### Issue 3: Documents Not Uploading

**Problem**: Documents upload fail ஆகிறது.

**Solutions**:
- File size limit check செய்யவும் (usually 2MB)
- File format (PDF/JPG/PNG) correct ஆக இருக்கிறதா verify செய்யவும்
- Internet connection stable ஆக இருக்கிறதா check செய்யவும்
- Different browser try செய்யவும்
- File compress செய்து upload செய்யவும்

### Issue 4: Payment Failed

**Problem**: Payment attempt செய்த பிறகு failed ஆகிறது.

**Solutions**:
- Bank account/credit card இல் sufficient balance இருக்கிறதா check செய்யவும்
- Internet connection stable ஆக இருக்கிறதா verify செய்யவும்
- Payment gateway issue இருந்தால், few minutes கழித்து retry செய்யவும்
- Different payment method try செய்யவும்
- Bank உடன் தொடர்பு கொண்டு transaction status check செய்யவும்

### Issue 5: Application Status Not Updating

**Problem**: Application status update ஆகவில்லை.

**Solutions**:
- Application number note செய்யவும்
- Processing time wait செய்யவும்
- Portal இல் status check செய்யவும்
- Helpdesk உடன் application number share செய்து status check செய்யவும்
- Email/SMS updates check செய்யவும்
