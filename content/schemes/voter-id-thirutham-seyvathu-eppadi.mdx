## Voter ID திருத்தம் செய்வது எப்படி?

Voter ID திருத்தம் செய்வது என்பது முக்கியமான செயல்முறையாகும். Name spelling mistakes, address change, photo update போன்ற திருத்தங்கள் தேவைப்படலாம். இந்த கட்டுரையில், Voter ID திருத்தம் செய்வதற்கான முழுமையான வழிமுறைகளை பார்க்கலாம்.

## Voter ID திருத்தம் - முக்கியத்துவம்

### ஏன் திருத்தம் தேவை?

- **Name Correction**: Spelling mistakes, name change
- **Address Update**: Current address update (constituency change)
- **Photo Update**: Old photo replace, better quality photo
- **Voting Rights**: Accurate information for voting
- **Legal Compliance**: Correct voter information

## Voter ID திருத்தம் வகைகள்

### 1. Name Correction

Name spelling mistakes, name change, அல்லது name update செய்யலாம்.

**Examples**:
- "Kumar" → "Kumaran"
- "Priya" → "Priyanka"
- Spelling mistakes correction

### 2. Address Change

Current address update செய்யலாம். New constituency change செய்யலாம்.

**Examples**:
- Old Address: 123, Main Street, Chennai - 600001
- New Address: 456, Park Avenue, Chennai - 600040
- Constituency change if address in different constituency

### 3. Photo Update

Old photo replace செய்யலாம். Recent, clear photo தேவை.

## Voter ID திருத்தம் செயல்முறை

### முறை 1: Online (NVSP Portal)

NVSP (National Voters' Service Portal) மூலம் online திருத்தம் செய்யலாம்.

#### Step-by-Step Procedure:

1. **Portal Visit**
   - Browser இல் [https://www.nvsp.in](https://www.nvsp.in) visit செய்யவும்
   - அல்லது [https://voterportal.eci.gov.in](https://voterportal.eci.gov.in) visit செய்யவும்

2. **Registration/Login**
   - New user என்றால், "Register" click செய்யவும்
   - Existing user என்றால், "Login" click செய்யவும்
   - Mobile number, email, password enter செய்யவும்
   - OTP verify செய்யவும்

3. **Correction/Update Section**
   - "Form 8 - Correction of Entries" select செய்யவும்
   - "Apply Online" click செய்யவும்

4. **Voter ID Details Enter**
   - EPIC Number (Voter ID Number) enter செய்யவும்
   - Name enter செய்யவும்
   - Date of Birth enter செய்யவும்
   - Verify details

5. **Correction Type Select**
   - Correction type select செய்யவும்:
     - Name Correction
     - Address Change
     - Photo Update
     - Date of Birth Correction
     - Gender Correction

6. **New Details Enter**
   - Selected correction type க்கு ஏற்ப new details enter செய்யவும்
   - Example: Name correction → New name enter

7. **Documents Upload**
   - Required documents upload செய்யவும்:
     - Old Voter ID copy
     - Proof of correction (name proof, address proof, etc.)
     - Recent photo (photo update க்கு)
     - Supporting documents

8. **Application Submit**
   - Form details verify செய்யவும்
   - "Submit" click செய்யவும்
   - Application Reference Number (ARN) note செய்யவும்

9. **Application Status Check**
   - ARN மூலம் status check செய்யலாம்
   - Processing time: 15-30 working days

10. **Updated Voter ID Download**
    - Application approved ஆன பிறகு, updated Voter ID download செய்யலாம்
    - PDF format இல் download செய்யலாம்

### முறை 2: Offline (Electoral Registration Officer)

#### Step-by-Step Procedure:

1. **Office Locate**
   - Nearest Electoral Registration Officer (ERO) office locate செய்யவும்
   - District/Assembly constituency level office visit செய்யவும்

2. **Application Form Collect**
   - Form 8 (Correction of Entries) collect செய்யவும்
   - அல்லது [https://www.nvsp.in](https://www.nvsp.in) இல் download செய்யவும்

3. **Form Fill**
   - Form fill செய்யவும்:
     - EPIC Number
     - Name
     - Date of Birth
     - Correction Type
     - New Details
     - Contact Information

4. **Documents Prepare**
   - Required documents prepare செய்யவும்:
     - Original old Voter ID
     - Self-attested copy of old Voter ID
     - Proof of correction
     - Supporting documents
     - Recent photo (photo update க்கு)

5. **Office Visit**
   - Office visit செய்து, form மற்றும் documents submit செய்யவும்

6. **Verification**
   - Officer verification செய்யலாம்
   - Additional documents request செய்யலாம் (if needed)

7. **Acknowledgement Receipt**
   - Acknowledgement receipt collect செய்யவும்
   - Application Reference Number (ARN) note செய்யவும்

8. **Status Check**
   - ARN மூலம் status check செய்யலாம்
   - Processing time: 15-30 working days

9. **Updated Voter ID Collect**
   - Application approved ஆன பிறகு, office இல் updated Voter ID collect செய்யலாம்
   - அல்லது online download செய்யலாம்

## தேவையான ஆவணங்கள்

### Name Correction

1. **Old Voter ID Copy**
   - Original Voter ID
   - Self-attested copy

2. **Name Proof**
   - Aadhaar Card (with correct name)
   - PAN Card
   - Birth Certificate
   - School Certificate
   - Government ID with correct name

3. **Supporting Documents**
   - Affidavit (if name change)
   - Gazette notification (if name change)

### Address Change

1. **Old Voter ID Copy**
   - Original Voter ID
   - Self-attested copy

2. **Address Proof**
   - Electricity Bill (latest, not older than 3 months)
   - Gas Bill
   - Bank Statement/Passbook
   - Rent Agreement
   - Aadhaar Card (with new address)
   - Ration Card (with new address)

3. **Identity Proof**
   - Aadhaar Card
   - PAN Card
   - Driving License

### Photo Update

1. **Old Voter ID Copy**
   - Original Voter ID
   - Self-attested copy

2. **Recent Photo**
   - Recent passport size photo
   - White background
   - Clear and visible
   - Same as other government IDs (if possible)

3. **Identity Proof**
   - Aadhaar Card
   - PAN Card
   - Other government ID

## Voter ID திருத்தம் - Common Issues மற்றும் Solutions

### Issue 1: Application Rejected

**Problem**: Application submit செய்த பிறகு rejected ஆகிறது.

**Solutions**:
- Rejection reason check செய்யவும்
- Documents verify செய்யவும்
- Missing documents upload/submit செய்யவும்
- Corrected application resubmit செய்யவும்
- Helpdesk உடன் தொடர்பு கொள்ளவும்

### Issue 2: EPIC Number Not Found

**Problem**: EPIC Number enter செய்த பிறகு "Not Found" error வருகிறது.

**Solutions**:
- EPIC Number சரியாக enter செய்துள்ளீர்களா verify செய்யவும்
- Latest Voter ID இல் உள்ள EPIC Number use செய்யவும்
- Spaces அல்லது special characters இல்லாமல் enter செய்யவும்
- ERO office உடன் verify செய்யவும்

### Issue 3: Documents Not Uploading

**Problem**: Documents upload செய்யும்போது error வருகிறது.

**Solutions**:
- File size 2MB க்கு கீழே இருக்கிறதா check செய்யவும்
- File format (PDF/JPG/PNG) correct ஆக இருக்கிறதா verify செய்யவும்
- Internet connection stable ஆக இருக்கிறதா check செய்யவும்
- Different browser try செய்யவும்
- File compress செய்து upload செய்யவும்

### Issue 4: Application Status Not Updating

**Problem**: Application submit செய்த பிறகு status update ஆகவில்லை.

**Solutions**:
- ARN note செய்யவும்
- Few days wait செய்யவும் (processing time)
- Portal இல் status check செய்யவும்
- Helpdesk உடன் ARN share செய்து status check செய்யவும்
- Office visit செய்து status verify செய்யவும்

### Issue 5: Updated Voter ID Not Received

**Problem**: Application approved ஆன பிறகு updated Voter ID receive ஆகவில்லை.

**Solutions**:
- Portal இல் download செய்யவும்
- ARN மூலம் download செய்யவும்
- Offline application செய்திருந்தால், office இல் collect செய்யலாம்

### என்ன திருத்தலாம்?

Name, address, photo, date of birth, gender திருத்தலாம். Multiple corrections single application இல் செய்யலாம்.

### தேவையான ஆவணங்கள் என்ன?

Correction type பொறுத்து documents vary ஆகும்:
- **Name Correction**: Old Voter ID, name proof (Aadhaar/PAN/Birth Certificate), supporting documents
- **Address Change**: Old Voter ID, address proof (Electricity Bill/Gas Bill/Bank Statement), identity proof
- **Photo Update**: Old Voter ID, recent photo, identity proof

### எவ்வளவு நேரம் எடுக்கும்?

Processing time: 15-30 working days. ARN மூலம் status check செய்யலாம்.

### Fee எவ்வளவு?

Voter ID திருத்தம் completely free. No fees charged.
