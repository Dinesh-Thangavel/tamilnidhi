## Birth Certificate Online பெறுவதற்கான முறைகள்

### முறை 1: Tamil Nadu e-District Portal

Tamil Nadu e-District portal மூலம் Birth Certificate online apply செய்யலாம்.

#### Step-by-Step Procedure:

1. **Portal Visit**
   - Browser இல் [https://edistrict.tn.gov.in](https://edistrict.tn.gov.in) visit செய்யவும்

2. **Registration/Login**
   - New user என்றால், "Register" click செய்யவும்
   - Existing user என்றால், "Login" click செய்யவும்
   - Mobile number, email, password enter செய்யவும்
   - OTP verify செய்யவும்

3. **Service Selection**
   - "Birth Certificate" service select செய்யவும்
   - "Apply Online" click செய்யவும்

4. **Application Form Fill**
   - Child's name (குழந்தையின் பெயர்)
   - Date of birth (பிறந்த தேதி)
   - Place of birth (பிறந்த இடம்)
   - Gender (பாலினம்)
   - Father's name (தந்தையின் பெயர்)
   - Mother's name (தாயின் பெயர்)
   - Address (முகவரி)
   - Contact details

5. **Documents Upload**
   - Required documents upload செய்யவும்:
     - Hospital discharge certificate
     - Parent's identity proof (Aadhaar/PAN/Voter ID)
     - Parent's address proof
     - Marriage certificate (if applicable)

6. **Application Submit**
   - Form details verify செய்யவும்
   - "Submit" click செய்யவும்
   - Application number note செய்யவும்

7. **Application Status Check**
   - Application number மூலம் status check செய்யலாம்
   - Processing time: 7-15 working days

8. **Certificate Download**
   - Application approved ஆன பிறகு, certificate download செய்யலாம்
   - PDF format இல் download செய்யலாம்

### முறை 2: Chennai Corporation Portal

Chennai city இல் பிறந்த குழந்தைகளுக்கு, Chennai Corporation portal மூலம் apply செய்யலாம்.

#### Step-by-Step Procedure:

1. **Portal Visit**
   - [https://www.chennaicorporation.gov.in](https://www.chennaicorporation.gov.in) visit செய்யவும்

2. **Birth Certificate Section**
   - "Online Services" → "Birth Certificate" select செய்யவும்

3. **Application Form**
   - Required details fill செய்யவும்
   - Documents upload செய்யவும்
   - Submit செய்யவும்

4. **Certificate Download**
   - Application approved பிறகு download செய்யலாம்

## Birth Certificate Online Application - Required Documents

### Mandatory Documents

1. **Hospital Discharge Certificate**
   - Hospital இல் பிறந்த குழந்தைகளுக்கு
   - Hospital name, date of birth, parent's name details

2. **Parent's Identity Proof**
   - Aadhaar Card
   - PAN Card
   - Voter ID
   - Driving License
   - Passport

3. **Parent's Address Proof**
   - Aadhaar Card (address proof ஆக)
   - Electricity Bill
   - Gas Bill
   - Bank Statement
   - Rent Agreement
   - Voter ID

4. **Marriage Certificate** (if applicable)
   - Parents marriage certificate
   - Optional, ஆனால் helpful

### Document Format Requirements

- **File Format**: PDF, JPG, PNG
- **File Size**: Maximum 2MB per document
- **Quality**: Clear and readable
- **Language**: English or Tamil
