## ஆதார் இணைப்பு தேவைப்படும் இடங்கள்

### Bank Accounts

Bank accounts உடன் ஆதார் இணைப்பு கட்டாயம்:

- **Savings Account**: Savings account KYC
- **Current Account**: Current account KYC
- **Fixed Deposit**: FD account KYC
- **Recurring Deposit**: RD account KYC
- **Loan Accounts**: Loan account KYC

#### Bank Account Aadhaar Link - Step-by-Step:

1. **Bank Visit/Online**
   - Bank branch visit செய்யவும்
   - அல்லது Internet Banking/App மூலம் online link செய்யலாம்

2. **Aadhaar Linking Form**
   - Aadhaar linking form fill செய்யவும்
   - அல்லது online form fill செய்யவும்

3. **Aadhaar Details Enter**
   - 12-digit Aadhaar number enter செய்யவும்
   - Name as per Aadhaar enter செய்யவும்

4. **Documents Submit**
   - Aadhaar card copy submit செய்யவும்
   - Self-attested copy (offline)

5. **Biometric Verification** (if required)
   - Fingerprint scan செய்யலாம்
   - Iris scan செய்யலாம்

6. **Link Complete**
   - Link successful message
   - Account updated

### Mobile Numbers

Mobile numbers உடன் ஆதார் இணைப்பு கட்டாயம்:

- **SIM Card**: New SIM card activation
- **Existing SIM**: Existing SIM re-verification
- **Multiple SIMs**: Multiple SIM cards

#### Mobile Number Aadhaar Link - Step-by-Step:

1. **Telecom Operator**
   - Airtel, Jio, Vi, BSNL போன்ற operator select செய்யவும்

2. **Store Visit/Online**
   - Store visit செய்யவும்
   - அல்லது online portal/app மூலம் link செய்யலாம்

3. **Aadhaar Details Enter**
   - 12-digit Aadhaar number enter செய்யவும்
   - Mobile number enter செய்யவும்

4. **Biometric Verification**
   - Fingerprint scan செய்யவும்
   - Iris scan செய்யவும் (if required)

5. **OTP Verification**
   - Registered mobile number இல் OTP receive செய்யவும்
   - OTP enter செய்து verify செய்யவும்

6. **Link Complete**
   - Link successful message
   - SIM activated/verified

### PAN Card

PAN card உடன் ஆதார் இணைப்பு கட்டாயம்:

- **Income Tax Filing**: ITR filing க்கு தேவை
- **Financial Transactions**: High-value transactions
- **Compliance**: Tax compliance

#### PAN-Aadhaar Link Procedure:

1. **Income Tax Portal**
   - [https://www.incometax.gov.in](https://www.incometax.gov.in) visit செய்யவும்

2. **Link Aadhaar**
   - "Profile" → "Link Aadhaar" select செய்யவும்

3. **Details Enter**
   - PAN number enter செய்யவும்
   - Aadhaar number enter செய்யவும்
   - Name, DOB enter செய்யவும்

4. **OTP Verify**
   - OTP verify செய்யவும்

5. **Link Complete**
   - Link successful

## ஆதார் இணைப்பு - Common Issues மற்றும் Solutions

### Issue 1: Name Mismatch

**Problem**: Aadhaar இல் name, service இல் name match ஆகவில்லை.

**Solutions**:
- Name exactly as per Aadhaar use செய்யவும்
- Aadhaar name correction செய்யலாம் (if needed)
- Service name correction செய்யலாம் (if needed)
- Supporting documents provide செய்யலாம்

### Issue 2: Biometric Verification Failed

**Problem**: Biometric verification fail ஆகிறது.

**Solutions**:
- Fingerprint clearly place செய்யவும்
- Finger clean ஆக இருக்க வேண்டும்
- Multiple attempts try செய்யவும்
- Different finger try செய்யவும்
- Iris scan try செய்யலாம் (if available)

### Issue 3: OTP Not Received

**Problem**: OTP request செய்த பிறகு OTP receive ஆகவில்லை.

**Solutions**:
- Registered mobile number verify செய்யவும்
- Mobile network signal check செய்யவும்
- SMS inbox full ஆக இருக்கிறதா check செய்யவும்
- Few minutes wait செய்து retry செய்யவும்
- "Resend OTP" option use செய்யவும்

### Issue 4: Link Already Exists

**Problem**: Already linked message வருகிறது.

**Solutions**:
- Link status verify செய்யவும்
- Already linked ஆக இருந்தால், no action needed
- Different account/number use செய்யலாம் (if applicable)
