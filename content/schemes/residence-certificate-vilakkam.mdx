## Residence Certificate பெறுவதற்கான முறைகள்

### முறை 1: Online (Tamil Nadu e-District Portal)

Tamil Nadu e-District portal மூலம் Residence Certificate online apply செய்யலாம்.

#### Step-by-Step Procedure:

1. **Portal Visit**
   - Browser இல் [https://edistrict.tn.gov.in](https://edistrict.tn.gov.in) visit செய்யவும்

2. **Registration/Login**
   - New user என்றால், "Register" click செய்யவும்
   - Existing user என்றால், "Login" click செய்யவும்
   - Mobile number, email, password enter செய்யவும்
   - OTP verify செய்யவும்

3. **Service Selection**
   - "Residence Certificate" service select செய்யவும்
   - "Apply Online" click செய்யவும்

4. **Application Form Fill**
   - Personal details enter செய்யவும்:
     - Name (பெயர்)
     - Date of Birth (பிறந்த தேதி)
     - Gender (பாலினம்)
     - Father's Name (தந்தையின் பெயர்)
     - Mother's Name (தாயின் பெயர்)
     - Current Address (தற்போதைய முகவரி)
     - Permanent Address (நிரந்தர முகவரி)
     - Contact details
   - Residence details enter செய்யவும்:
     - Residence Period (வசிப்பு காலம்)
     - Residence Type (Permanent/Temporary)

5. **Documents Upload**
   - Required documents upload செய்யவும்:
     - Aadhaar Card (with current address)
     - Voter ID Card (with current address)
     - Ration Card (with current address)
     - Electricity Bill (latest, not older than 3 months)
     - Gas Bill
     - Bank Statement (with current address)
     - Rent Agreement (if rented)
     - Property Documents (if owned)

6. **Application Submit**
   - Form details verify செய்யவும்
   - "Submit" click செய்யவும்
   - Application number note செய்யவும்

7. **Application Status Check**
   - Application number மூலம் status check செய்யலாம்
   - Processing time: 7-15 working days

8. **Certificate Download**
   - Application approved ஆன பிறகு, certificate download செய்யலாம்
   - PDF format இல் download செய்யலாம்

### முறை 2: Offline (Tahsildar Office)

Online apply செய்ய முடியாத நிலையில், offline மூலம் Residence Certificate பெறலாம்.

#### Step-by-Step Procedure:

1. **Office Locate**
   - Nearest Tahsildar office locate செய்யவும்
   - Taluk/District level office visit செய்யவும்

2. **Application Form Collect**
   - Residence Certificate application form collect செய்யவும்
   - அல்லது [https://edistrict.tn.gov.in](https://edistrict.tn.gov.in) இல் download செய்யவும்

3. **Form Fill**
   - Form fill செய்யவும்:
     - Personal details
     - Residence details
     - Address details
     - Contact information

4. **Documents Prepare**
   - Required documents prepare செய்யவும்:
     - Original Aadhaar Card (with current address)
     - Self-attested copy of Aadhaar Card
     - Voter ID Card copy (with current address)
     - Ration Card copy (with current address)
     - Electricity Bill (latest, not older than 3 months)
     - Gas Bill
     - Bank Statement (with current address)
     - Rent Agreement (if rented)
     - Property Documents (if owned)

5. **Office Visit**
   - Office visit செய்து, form மற்றும் documents submit செய்யவும்

6. **Verification**
   - Officer verification செய்யலாம்
   - Additional documents request செய்யலாம் (if needed)
   - Field verification செய்யப்படலாம்

7. **Acknowledgement Receipt**
   - Acknowledgement receipt collect செய்யவும்
   - Application number note செய்யவும்

8. **Status Check**
   - Application number மூலம் status check செய்யலாம்
   - Processing time: 7-15 working days

9. **Certificate Collect**
   - Application approved ஆன பிறகு, office இல் certificate collect செய்யலாம்
   - அல்லது online download செய்யலாம்

## Residence Certificate - Common Issues மற்றும் Solutions

### Issue 1: Application Rejected

**Problem**: Application submit செய்த பிறகு rejected ஆகிறது.

**Solutions**:
- Rejection reason check செய்யவும்
- Documents verify செய்யவும்
- Missing documents upload/submit செய்யவும்
- Address details accurate ஆக enter செய்துள்ளீர்களா verify செய்யவும்
- Latest address proof use செய்யவும் (not older than 3 months)
- Corrected application resubmit செய்யவும்
- Helpdesk உடன் தொடர்பு கொள்ளவும்

### Issue 2: Address Proof Not Accepted

**Problem**: Address proof documents rejected ஆகிறது.

**Solutions**:
- Latest documents use செய்யவும் (not older than 3 months)
- All address proof documents consistent ஆக இருக்க வேண்டும்
- Multiple address proofs submit செய்யவும்
- Electricity bill, gas bill, bank statement போன்ற multiple proofs use செய்யவும்

### Issue 3: Field Verification

**Problem**: Field verification request செய்யப்படுகிறது.

**Solutions**:
- Verification officer cooperate செய்யவும்
- All documents ready வைத்திருங்கள்
- Residence details accurately explain செய்யவும்
- Neighbors verification cooperate செய்யவும் (if required)
